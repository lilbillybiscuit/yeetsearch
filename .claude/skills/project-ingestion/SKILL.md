# Skill: project-ingestion

Use when testing the framework against an arbitrary project.

1. Read root instructions such as `AGENTS.md`, `CLAUDE.md`, or README files.
2. Record source layout, build metadata, and test metadata.
3. Capture git commit if available; otherwise record the failure.
4. Do not edit the target project during ingestion.
5. Write a snapshot JSON artifact for downstream agents.
